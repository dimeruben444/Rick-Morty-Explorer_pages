#  Rick & Morty Explorer 

Este es un ejemplo de explorador aplicando conocimientos en Javascript 

Objetivo: Crear una aplicación web que consuma datos de la API de Rick and Morty y permita al usuario 
explorar personajes, episodios o localizaciones de la serie. Se ha implementado "Vite" como entorno de desarrollo, "Sass" como preprocesador CSS y Javascript carga dinámica desde API, buenas prácticas de estructura HTML y organización de carpetas asi como un diseño responsive web desing para tablet movil y desktop.

##  Tecnologías usadas

- HTML5 semántico
- Javascript carga dinámica desde API
- Sass (SCSS) para estilos avanzados
- Vite como bundler moderno
- Git y GitHub Pages para despliegue

##  Objetivos

- Búsqueda y Filtrado de Personajes
- Paginación o Carga Dinámica
- Favoritos con LocalStorage
- Manejo de Errores y Estados
- Incluir navegación intuitiva
- Estilizar con variables, mixins y componentes Sass
- Aplicar estilos responsive mobile first 
- Publicar el proyecto en GitHub Pages

##  Aprendizaje

Este proyecto me ha permitido afianzar el uso de JavaScript para consumir una API y cargar datos de forma dinámica en el navegador, mejorando mi comprensión del flujo asíncrono y la manipulación del DOM. También he consolidado el uso de Sass, bajo un enfoque mobile first y una estructura de estilos más modular. Además, he profundizado en el flujo de trabajo con Vite como entorno de desarrollo moderno, he reforzado el uso de Git para el control de versiones y he desplegado el proyecto con éxito en GitHub Pages.
